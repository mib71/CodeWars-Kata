## 7 kyu Simple equation reversal

### Details:
Given a mathematical equation that has *,+,-,/, reverse it as follows:  
 ```
solve("100*b/y") = "y/b*100"
solve("a+b-c/d*30") = "30*d/c-b+a"
 ```
