## 7 kyu Return the first M multiples of N

### Instructions

Implement a function, multiples(m, n), which returns an array of the  
first m multiples of the real number n. Assume that m is a positive integer.

### Example
```
multiples(3, 5.0)
```
### should return
```
[5.0, 10.0, 15.0]
```
